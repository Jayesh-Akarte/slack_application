# Messaging Application

This is chat like stack interface.
This is a messaging application.

## Features Implemented

### 1. Login Screen
- Email and password input
- Navigation to channel screen after login

### 2. Channel List Screen
- Displays list of channels
- Shows unread message count
- Tap to open messages for selected channel

### 3. Message Screen
- Displays list of messages
- Basic chat bubble UI
- Message input field at bottom
- Send button UI


## State Management

Currently using StatefulWidget for managing local UI state.  
Messages are stored in simple lists.
